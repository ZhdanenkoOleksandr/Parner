## 🌐 О проекте

Web 4.0 Bitbon System — это платформа, объединяющая людей, бизнес и капитал на основе новой культуры взаимодействия, построенной на доверии, прозрачности и осознанном участии.

**Ключевые принципы:**
- ✅ Честность и открытость в коммуникации
- 🤝 Ответственность и целостность в выполнении обязательств
- 💎 Цифровая репутация как ценный актив
- 🌍 Устойчивые цифровые и экономические модели

## 🚀 Быстрый старт

### Предварительные требования

- Node.js 18.0 или выше
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/web4-bitbon-system.git
cd web4-bitbon-system
```

2. Установите зависимости:
```bash
npm install
# или
yarn install
```

3. Запустите development сервер:
```bash
npm run dev
# или
yarn dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## 📦 Развертывание на Vercel

### Способ 1: Через GitHub (рекомендуется)

1. Загрузите код на GitHub
2. Зайдите на [vercel.com](https://vercel.com)
3. Нажмите "New Project"
4. Импортируйте ваш репозиторий
5. Vercel автоматически определит Next.js и настроит проект
6. Нажмите "Deploy"

### Способ 2: Через Vercel CLI

```bash
# Установите Vercel CLI
npm i -g vercel

# Разверните проект
vercel

# Для production
vercel --prod
```
