## 🎨 Что можно делать прямо сейчас?

1. **Посмотреть работу приложения**
   ```bash
   npm run dev
   ```

2. **Изменить контент**
   - Откройте `pages/index.js`
   - Найдите нужную секцию
   - Отредактируйте текст

3. **Настроить контакты**
   - В `pages/index.js` найдите секцию ContactPage
   - Замените на свои данные

4. **Загрузить на GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/username/repo.git
   git push -u origin main
   ```

5. **Развернуть на Vercel**
   - Зайдите на vercel.com
   - Нажмите "New Project"
   - Выберите ваш репозиторий
   - Нажмите "Deploy"
