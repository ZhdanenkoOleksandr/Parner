Спасибо за ваш интерес к проекту! Мы приветствуем участие единомышленников, которые разделяют наши ценности.

- **Ответственность** — выполнение взятых на себя обязательств

## 🤝 Как принять участие

### 1. Предложить улучшение

Если у вас есть идея по улучшению:
- Откройте Issue с описанием предложения
- Опишите проблему и предлагаемое решение
- Добавьте примеры, если возможно

### 2. Исправить баг

Нашли ошибку?
- Создайте Issue с описанием бага
- Укажите шаги для воспроизведения
- Опишите ожидаемое поведение

### 3. Внести код

Хотите добавить функционал?
1. Создайте fork репозитория
2. Создайте feature branch (`git checkout -b feature/AmazingFeature`)
3. Сделайте изменения с понятными коммитами
4. Push в ваш fork (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## 📝 Стандарты кода

- Используйте осмысленные имена переменных и функций
- Комментируйте сложную логику
- Следуйте существующему стилю кода
- Тестируйте изменения локально перед PR
